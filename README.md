# Gerador de Senha Segura

Um gerador de senha moderno, elegante e responsivo construído com HTML, CSS e JavaScript puros. Este projeto cria senhas fortes com opções de personalização, indicador de força visual e um estilo escuro sofisticado inspirado em temas de segurança e criptografia.

## ✨ Recursos

- Geração de senhas aleatórias com comprimento configurável
- Opções para incluir:
  - Maiúsculas (A-Z)
  - Minúsculas (a-z)
  - Números (0-9)
  - Símbolos especiais (!@#$)
- Indicador visual de força da senha
- Botão de cópia para transferir a senha para a área de transferência
- Layout responsivo e tema escuro
- Background animado com estética de criptografia

## 📁 Estrutura do Projeto

- `index.html` — Estrutura principal da página e elementos de interface
- `style.css` — Estilos, animações e visual responsivo do layout
- `index.js` — Lógica de geração de senha, gerenciamento de eventos e comportamento da interface

## 🚀 Como usar

1. Abra `index.html` no navegador.
2. Ajuste o `Comprimento` usando o controle deslizante.
3. Marque as opções de caracteres desejadas:
   - Maiúsculas
   - Minúsculas
   - Números
   - Símbolos
4. Clique em **Gerar Senha**.
5. Clique no ícone de cópia para copiar a senha gerada.
